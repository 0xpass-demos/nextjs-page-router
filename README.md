# Getting Started with Create Next App

This project was bootstrapped with Create Next App

## Install the project

In the project directory, you can run:


```bash
pnpm install
```


or

```bash
yarn install
```

or


```bash
npm install --force
```


## Signup on 0xpass

Signup on [0xpass Dashboard](https://dashboard.0xpass.io/)
